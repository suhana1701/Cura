# 🩺 Cura

Cura is a modern web-based healthcare/fitness application designed to help users manage their wellness, track activities, and improve overall health through a clean and intuitive interface.

---

## 🚀 Features

- 🧑‍⚕️ User-friendly healthcare/fitness interface  
- 📊 Dashboard for tracking activities and progress  
- 📝 Manage health-related data efficiently  
- 📱 Fully responsive design (mobile + desktop)  
- ⚡ Fast performance with optimized frontend
